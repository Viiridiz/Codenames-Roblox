# Codenames Project (Roblox/Knit)

## Setup Instructions
1. **Install Tools:**
   - Install VS Code.
   - Install [Aftman](https://github.com/LPGhatguy/aftman).
   - Run `aftman install` in this terminal (installs Rojo and Wally).

2. **Initialize Project:**
   - Run `wally install` (Downloads Knit/Roact).
   - Run `rojo build -o Codenames.rbxlx`.

3. **Develop:**
   - Open `Codenames.rbxlx` in Roblox Studio.
   - Run `rojo serve` in VS Code to sync changes.

## Architecture
- **Framework:** Knit (Service/Controller)
- **UI:** Roact
- **Testing:** TestEZ (Run tests by pressing Play in Studio)
